Highest-value builds for Week 2

  • Onboarding trigger agentOn PandaDoc contract sign: auto-create HiBob profile, trigger Zinc, save contract to HiBob, raise IT Slack ticket. Removes ~30 min of Jess's time per hire, fully automated.
  • Pre-boarding comms agent1 week out: auto-send HiBob task list (bank details, allergies), create Ben account, schedule department intros and Amon intro from a calendar template.
  • Day 1 benefits emailAuto-send Vitality and Medicash info on start date. Templated, no input needed from Jess.
  • Compliance monitorChecks HiBob and Drata daily, sends automated chasers, surfaces a completion report to Jess at month 1.
  • Probation pre-read agentAt the 3-month mark, compiles goals progress, Drata completion, and HiBob activity into a one-page pre-read for the check-in meeting.

Known blockers to resolve in Week 1

  • Greenhouse → HiBob: No MCP until Q3. Confirm whether PandaDoc webhook alone is sufficient as the onboarding trigger, or if Greenhouse API is needed.
  • IT account provisioning: Confirm with IT whether Google Workspace and Slack have admin API access for automated account creation.
  • Zinc: MCP not yet connected. Assess whether Zinc API supports programmatic background check triggering.
  • Ben: Confirm whether Ben has an API or admin bulk-invite flow that an agent can use.
